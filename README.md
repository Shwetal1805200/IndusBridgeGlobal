# IndusBridge Global: The Sovereign Ledger

A high-end, editorial-style web application for **IndusBridge Global**, India's distribution-first export partner for GCC distributors. This modernization transforms static trade documentation into a high-trust, responsive digital experience.

## ✨ Features

- **The Sovereign Ledger Design System**: A premium, authoritative aesthetic utilizing tonal layering, glassmorphism, and technical-ledger inspired typography.
- **Modern SPA Framework**: Built with a lightweight hash-based router for seamless navigation across all vertical trade disciplines.
- **Engineered Interactions**:
  - **GSAP & ScrollTrigger**: Custom scroll-driven reveals and micro-animations.
  - **Lenis**: Integrated smooth scrolling for a premium, buttery feel.
  - **SplitType**: High-impact editorial headline animations.
- **Fully Responsive**: Optimized for every jurisdiction from desktop terminals to mobile handsets.
- **Lead-Generation Architecture**: Functional inquiry flows that bridge distributors to global logistics data.

## 🏗️ Technology Stack

- **Core**: Vanilla JavaScript, Semantic HTML5
- **Styling**: Vanilla CSS (Custom Design System)
- **Tooling**: Vite (Build System)
- **Libraries**:
  - [GSAP](https://gsap.com/) (Animations)
  - [Lenis](https://github.com/darkroomengineering/lenis) (Smooth Scroll)
  - [SplitType](https://github.com/lukePeavey/SplitType) (Typography)

## 🚀 Getting Started

To run the application locally:

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```
